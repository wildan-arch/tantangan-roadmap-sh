# Single-Page CV (HTML Semantik)

Proyek ini adalah implementasi pembuatan _Curriculum Vitae_ (CV) sederhana berbasis web satu halaman yang berfokus sepenuhnya pada penggunaan **Semantic HTML**. Proyek ini dibuat sebagai bagian dari latihan dasar untuk memahami struktur kode web yang baik, ramah SEO, dan aksesibel.

## 🚀 Fitur & Keunggulan Kode

- **100% Semantic HTML:** Menggunakan elemen terstruktur seperti `<header>`, `<main>`, `<section>`, `<article>`, `<address>`, `<nav>`, dan `<footer>` untuk memastikan kode memiliki arti terstruktur.
- **Aksesibilitas & SEO Friendly:** Penataan tag heading (`<h1>` hingga `<h3>`) yang hierarkis dan penggunaan tag `<time>` untuk penulisan riwayat tahun.
- **Direct Contact Links:** Fitur integrasi tautan langsung untuk mengirim email (`mailto:`), melakukan panggilan telepon (`tel:`), serta navigasi media sosial aktif.
- **Meta Tag Terkonfigurasi:** Sudah dilengkapi dengan meta deskripsi, kata kunci (_keywords_), penentuan _author_, serta ikon (_favicon_).
