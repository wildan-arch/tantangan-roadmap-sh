# 🚀 Shabah Cloud Hosting - Semantic HTML & Accessible Table

Proyek sederhana ini berisi implementasi struktur web semantik HTML5 dan tabel perbandingan paket layanan yang ramah aksesibilitas (_Accessible Web Table_). Proyek ini dibuat sebagai bentuk latihan praktis dalam menerapkan standar penulisan kode HTML modern.

---

## 📌 Fitur & Keunggulan Kode

Halaman ini dibangun dengan memperhatikan standar **W3C** dan **Aksesibilitas Web (WCAG)**:

- **Struktur Semantik Utuh:** Menggunakan tag `<header>`, `<main>`, dan `<footer>` sebagai fondasi utama halaman.
- **Tabel Aksesibel (Ramah Disabilitas):**
  - Pemisahan struktur data menggunakan `<thead>` dan `<tbody>`.
  - Judul tabel diletakkan menggunakan tag `<caption>` di dalam `<table>`.
  - Penerapan atribut `scope="col"` pada baris header kolom dan `scope="row"` pada baris header data untuk mendukung alat pembaca layar (_screen reader_).
- **Penggabungan Sel (_Merged Cell_):** Menggunakan atribut `colspan="4"` untuk membuat sekat kategori fitur tambahan.
- **Metadata Lengkap:** Dilengkapi dengan konfigurasi `<meta charset>`, `<meta viewport>`, deskripsi SEO, serta penulisan `author`.
